
## التحسينات الأساسية للمنصة 🚀

### 1️⃣ نظام إشعارات Push (Firebase/OneSignal)
- إشعارات فورية للسائقين عند وصول طلب جديد
- إشعارات للعملاء عند تحديث حالة الطلب/الرحلة
- إشعارات للمدير عند شكوى أو تذكرة جديدة

### 2️⃣ إشعارات واتساب
- رابط واتساب مباشر (بدون API) للتواصل السريع
- زر "شارك عبر واتساب" للإحالات
- إرسال رابط واتساب تلقائي عند تأكيد الطلب

### 3️⃣ نظام استعادة كلمة المرور بالإيميل
- تحسين صفحة "نسيت كلمة المرور"
- إرسال رابط إعادة تعيين عبر البريد الإلكتروني
- صفحة إعادة تعيين كلمة المرور الجديدة

### 4️⃣ تقارير مالية PDF
- تصدير تقارير الأرباح للسائقين
- تقارير يومية/أسبوعية/شهرية
- تقارير العمولات للمدير

### 5️⃣ تحسينات UX إضافية
- مؤشر حالة الاتصال بالإنترنت
- رسائل خطأ أوضح وأكثر ودية
- تحسين سرعة التحميل
