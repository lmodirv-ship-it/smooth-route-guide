
الهدف: إصلاح مسار النشر/التحديث حتى تظهر آخر التعديلات فعلياً على المنصة.

ما وجدته الآن
- سكربت السيرفر `scripts/server/auto-deploy.sh` يستخدم `npm install --legacy-peer-deps` بينما `package.json` يفرض `pnpm` عبر `preinstall`؛ وهذا سبب مباشر محتمل لفشل التحديث.
- نفس السكربت يمرر أوامر البناء عبر `tail` بدون `pipefail`، لذلك قد يتم إخفاء الخطأ ويبدو كأن البناء نجح.
- بناء HN-Stock ينتج ملف دخول باسم `hn-stock.html` داخل `dist-hn-stock/`، بينما بعض إعدادات الخادم/الوثائق تتوقع `index.html`؛ هذا يسبب تعارضاً بعد النشر.
- الرابط المنشور الحالي لا يعكس واجهة التطبيق المتوقعة، لذلك يلزم ضبط مسار النشر العام أيضاً بعد إصلاح pipeline.

خطة الإصلاح
1. توحيد أداة التثبيت والبناء
- تعديل `auto-deploy.sh` لاستخدام `pnpm install --frozen-lockfile` وسكربتات `pnpm build:*` بدلاً من `npm` و`npx`.
- إضافة `set -euo pipefail` حتى يفشل السكربت بشكل واضح عند أي خطأ.

2. توحيد ملف الدخول لـ HN-Stock
- اعتماد مسار واحد ثابت:
  - إما إخراج `index.html`
  - أو الإبقاء على `hn-stock.html` مع تحديث كل إعدادات الاستضافة والنسخ لتتوافق معه
- سأوحد هذا في كل الأماكن حتى لا يبقى تعارض بين البناء وNginx والسيرفر.

3. مزامنة ملفات النشر المرتبطة
- تحديث الملفات المرتبطة بمسار النشر:
  - `.github/workflows/deploy.yml`
  - `DEPLOYMENT.md`
  - `scripts/server/setup-server.sh`
  - `scripts/server/setup-hn-stock.sh`

4. التحقق من النشر العام
- بعد الإصلاح، يتم نشر الواجهة من خلال `Publish > Update`.
- ثم التحقق أن رابط `.lovable.app` يعرض آخر نسخة، وأن `hn-driver.site` يستقبل البناء الجديد بدون فشل.

5. اختبار نهائي
- التأكد من:
  - نجاح التثبيت والبناء بدون أخطاء صامتة
  - ظهور آخر تعديلات HN-Stock
  - عمل الروابط المباشرة وRefresh
  - عدم بقاء السيرفر على نسخة قديمة

تفاصيل تقنية
- سبب المشكلة المرجح في pipeline نفسه، وليس في الواجهة فقط:
  - المشروع يرفض `npm install`
  - سكربت النشر ما زال يستخدم `npm`
  - ملف الدخول لـ HN-Stock غير موحد مع بعض إعدادات الاستضافة
- لذلك الضغط على “Publish/Update” وحده لن يحل المشكلة بالكامل قبل إصلاح هذه النقاط.

بعد الموافقة سأقوم بـ
- إصلاح سكربتات النشر
- توحيد entry point لـ HN-Stock
- تحديث ملفات الإعداد/التوثيق المرتبطة
- ثم إعطائك خطوة النشر النهائية المختصرة جداً للتحقق
